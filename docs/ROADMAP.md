# IA Trabalhista Robusta — Roadmap

## Fase 1 (Base robusta)
- Postgres + Alembic
- Users persistidos + RBAC
- Audit log em tabela
- Health + smoke

## Fase 2 (Coração do produto)
- Cases (cadastro do caso)
- Intake guiado -> JSON estruturado
- Timeline do caso
- Checklist de provas e lacunas

## Fase 3 (Produção de peças)
- Templates versionados
- Export Docx/PDF
- Anexos: upload + tags + vínculo com fatos

## Fase 4 (IA controlada)
- Modo IA on/off + anonimização
- Biblioteca do escritório (cláusulas)
- Guardrails: não inventar fatos
